Sobre o Projeto

Guiné-Bissau: Descubra o Paraíso Escondido

Este site foi desenvolvido como parte de um projeto acadêmico para a disciplina de Desenvolvimento Web, com o objetivo de criar um portal informativo e visualmente atraente sobre o turismo na Guiné-Bissau.

A Guiné‑Bissau é um país de grande riqueza natural e cultural, ainda pouco explorado turisticamente. Através deste projeto, buscamos divulgar seus principais destinos, tradições e a importância do turismo sustentável para o desenvolvimento local.

Objetivos do Projeto
  • ✔ Criar um site com no mínimo 5 páginas utilizando HTML5, CSS3 e Bootstrap.
  • ✔ Aplicar componentes prontos do Bootstrap (navbar, cards, carrossel, grid, modais).
  • ✔ Utilizar controle de versão com Git e publicar o projeto no GitHub Pages.
  • ✔ Elaborar documentação acadêmica detalhada com mais de 20 páginas.
  • ✔ Promover o turismo sustentável na Guiné-Bissau através de conteúdo relevante.
